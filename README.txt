# CS143 Project 1 - Movie Database

Part 1A
--------
Sets up basic database which can fulfill queries via PHP.
Demonstrates primary keys, referential keys, checks, basic SELECT query statements.

Files included: 
	create.sql - creates tables for database
	load.sql - loads data into each table of database
	queries.sql - MySQL SELECT statements that query the database
	query.php - web query interface
	violate.sql - various error messages when violating certian constraints
